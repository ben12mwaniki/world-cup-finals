# World cup finals
This app reads FIFA men's finals data from [wikipedia](https://en.wikipedia.org/wiki/List_of_FIFA_World_Cup_finals) and writes it to a JSON file. It uses cheerio to scrap this data.

## Setup and Configuration
* Fork the repo and run `npm i` to install the dependencies
* Run `npm run start`

_Disclaimer: Ensure that you have permission to scrap data from a site before proceeding_
